# Rune Forge Mythos Image Prompt Pack

Use this pack to generate cohesive stock assets in ChatGPT image generation for Rune Forge.

## Canonical Art Direction
- Theme: mythic-cosmic, ritual geometry, sacred but respectful.
- Tone: epic, mysterious, luminous, high-contrast.
- Rule: all assets must be original compositions, no logos, no text, no copyrighted characters, no direct replication of existing real-world religious emblems.
- Shape-safe zone: keep center 45% of image mostly clear for sigil geometry overlay.
- Quality: crisp edges, clean alpha when transparency is requested, no compression artifacts.

## Output Specs
- Backgrounds: `2048x2048`, PNG, opaque.
- Overlay ornaments: `2048x2048`, PNG, transparent background.
- Texture layers: `2048x2048`, PNG, tile-safe where noted.
- UI accents: `1024x1024`, PNG, transparent background.

## Universal Prompt Prefix
Use this at the beginning of each prompt:

`Create an original fantasy-mythic artwork for a mobile app asset. No text, no watermark, no logos, no copyrighted characters, no real-world sacred symbol replicas. Keep the center composition clear for a geometric sigil overlay. Highly detailed, cinematic lighting, clean production quality.`

## Universal Negative Prompt
Use this at the end of each prompt:

`Avoid text, letters, signatures, watermarks, blurry edges, low detail, duplicate motifs, modern city elements, sci-fi machinery, guns, gore, and recognizable trademarked iconography.`

## Curation Checklist

- No text or watermark.
- No direct real-world sacred symbol replication.
- Matches pack palette and mood.
- Looks good behind bright and dark sigil stroke colors.
- Alpha edges are clean for overlays.
