# assets/

This folder contains visual assets that showcase the Yggdrasil Engine in action.

## preview.jpg / preview.png

The main hero image displayed at the top of the repository README.  
Shows a single creature generated with the public pipeline (sigil → ImprintKey → L-system + gifts).

## creatures/ (optional)

Additional example creatures generated with the engine.  
You can add more images here (e.g., `veylthar.jpg`, `lumora.png`) and reference them in the main README.

All images in this folder are generated using the exact same deterministic math as the engine itself.
