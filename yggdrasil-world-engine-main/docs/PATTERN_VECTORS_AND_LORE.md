// (H, K, D, S, L)

- **K** = Kolmogorov complexity proxy (0 = perfectly compressible, 1 = random)  
- **D** = Fractal dimension (1–3 range)  
- **S** = Symmetry score (0–1)  
- **L** = Length of shortest L-system generator

These five numbers **fully determine**:

- Base plane placement
- Visual appearance
- Metaphysical gifts
- Narrative role
