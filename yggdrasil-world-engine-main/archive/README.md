# Archive – Legacy & Historical Snapshots

The `archive/` folder is home to historical versions of key files and unused conceptual drafts relevant to the Yggdrasil World Engine. These artifacts capture the evolution of the engine and are preserved for legal, historical, and archival purposes. This is a treasure trove for understanding the intellectual journeys and stages that shaped the modern engine.

**You do NOT need these files to run the current engine.**

## Highlights of the Folder

### Historical Descriptions

1. `v0.1-legacy-cosmology.json`
   - **Date**: March 15, 2024  
   - **Description**: Earliest prototype of the cosmological map containing only 7 planes. Lacks the “Void” and “Divine Core” layers found in later models.
   - **Significance**: Establishes the foundational layers and transition rules for adjacent planes.

2. `rune-mapping-draft-v0.3.json`
   - **Date**: June 22, 2024  
   - **Description**: A developmental attempt at representing runes using bitmasks. This early draft didn’t yet adopt the final 9-bit hypercube structure.
   - **Significance**: Shows the experimentation that resulted in the distinctive White Wolf/Dark Wolf binary coding.

3. `SymDim.json`
   - **Date**: December 25, 2025  
   - **Description**: Documents a pipeline transforming 2D geometric structures into higher-dimensional representations. Inspired by supersymmetry and Adinkra diagrams.
   - **Integration**: Links to the `SigilForge` procedural tool for feeding preprocessed sigil images into this pipeline.

4. `sigil-forge.json`
   - **Date**: December 25, 2025  
   - **Description**: Describes a procedural tool for deterministic geometric sigil creation based on user seeds and mathematical symmetry systems.
